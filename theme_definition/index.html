This prototype is based the jQuery plugin QueryBuilder
It uses Bootstrap, jQuery & CSS3

A few modifications have been made on the original JavaScript library to match our needs.

  • We need only one type of "filter": "descriptor"
    So the dropdown list to select the filter has been removed
  • The input field to enter the value is placed before the operator dropdown list
  • A custom list of operators has been implemented:
    • Exact Search
    • Regular Expression
    • Inflected Form
    • Semantic

General notes

  • The "Output" section presents a "human readable" form of the theme. It is updated when a value is entered in one of the text inputs or when an operator changes
  • The "JSON Result" section presents what the JSON representation of the theme. The left part shows the JSON as it is generated by QueryBuilder. The right part shows a possible version matching (partially) our specifications.
